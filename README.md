# Pre-Install
- Install `starship`
- Install `nerdfont`, configure terminal to use `nerdfont`
- Install `wget`

# Quick Install
**N.B** restart your shell to load changes
```
wget https://raw.githubusercontent.com/ayumodark/starship/refs/heads/config/starship.toml -P ~/.config
```
