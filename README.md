# Pre-Install
- Install `starship`
- Install `nerdfont`, configure terminal to use `nerdfont`
- Install `wget`

# Quick Install
```
cd ~/.config && wget https://raw.githubusercontent.com/ayumodark/starship/refs/heads/config/starship.toml && echo "eval "$(starship init bash)"" >> ~/.bashrc && exec bash
```
