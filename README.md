# Pre-Install
- Install `starship`
- Install `nerdfont`, configure terminal to use `nerdfont`
- Install `wget`

# Quick Install
```
wget https://raw.githubusercontent.com/ayumodark/starship/refs/heads/config/starship.toml -P ~/.config && echo 'eval "$(starship init zsh)"' >> ~/.zshrc && exec zsh
```

# Config Details
starship is easy to configure, see their [configuration](https://starship.rs/config) guide for details
